HTML

    Lenguaje de marcado utilizado para estructurar contenido en la web.

CSS

    Lenguaje de hojas de estilo que controla la presentación de los documentos HTML.

JavaScript

    Lenguaje de programación que permite agregar interactividad a las páginas web.

Frontend

    Parte de una aplicación web que interactúa directamente con el usuario, combinando HTML, CSS y JavaScript.

Framework

    Conjunto de herramientas y convenciones que facilitan el desarrollo de software, como React, Angular y Vue.
